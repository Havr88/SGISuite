# Guía de Uso del Sistema

A continuación se detallan los pasos para comenzar a gestionar el inventario de manera eficiente.

## 1. Configuración Inicial (Primer Ingreso)

Acceda con las credenciales por defecto (`admin` / `admin`).

1.  Diríjase a **Configuración -> Institución**.
2.  Complete los datos de su organismo o empresa.
3.  Suba su logotipo (en formato PNG preferiblemente).
4.  Guarde los cambios para que se reflejen en los encabezados.

## 2. Definición de la Base del Inventario

Antes de cargar stock, es vital definir las categorías y departamentos:

- **Categorías:** Defina grupos como "Consumibles", "Papelería", "Herramientas".
- **Departamentos:** Registre a quiénes se les entrega material (Ej: "Recursos Humanos", "Vigilancia").

## 3. Registro de Artículos

En la sección **Artículos**, agregue cada elemento indicando su Unidad de Medida (UND, KG, LT) y su **Stock Mínimo**. Este último activará las alertas visuales en el panel principal cuando el inventario sea bajo.

## 4. Operaciones Diarias

### Registro de Ingresos
Cuando reciba mercancía nueva, use **Registrar Ingreso**. Esto sumará existencias de forma auditable, permitiendo dejar notas sobre el número de factura o procedencia.

### Registro de Entregas (Egresos)
Para retirar stock, use **Registrar Entrega**. Deberá indicar la cantidad, el departamento destino y el nombre de la persona que recibe. El sistema descontará automáticamente el stock.

## 5. Auditoría y Reportes

En **Historial Movimientos**, podrá ver la trazabilidad de cada operación. Puede usar el botón de **Imprimir** para generar un reporte físico del historial filtrado por el navegador.
