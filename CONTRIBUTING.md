# Guía para Contribuir

¡Gracias por tu interés en mejorar el **Sistema de Gestión de Inventario**! Este es un proyecto de software libre que busca la soberanía tecnológica.

## ¿Cómo puedes ayudar?

1.  **Reportar Errores:** Si encuentras un problema, abre un *Issue* en GitHub detallando los pasos para reproducirlo.
2.  **Sugerir Mejoras:** ¡Todas las ideas para nuevas funcionalidades son bienvenidas!
3.  **Enviar Código:** 
    - Haz un *Fork* del repositorio.
    - Crea una rama (*branch*) para tu mejora: `git checkout -b feature/nueva-mejora`.
    - Envía un *Pull Request* (PR) explicando tus cambios.

## Estilo de Código

- Sigue las normas de Python **PEP 8**.
- Documenta las funciones clave.
- Mantén las plantillas Jinja2 limpias y comentadas.
